**Please note: You will have to remove consumer.txt, producer_black.txt,
producer_red.txt and producer_white.txt from directory if you want to
create new files otherwise the program will continue to write in those
files where previously run

To run program:

  1. Open terminal and navigate to the directory of the file
  2. Type 'make' in the terminal
  3. type './main' in terminal
  4. Open consumer.txt, producer_black.txt, producer_red.txt and producer_white.txt to view outputs
